# Spaceship Shooting Game "SPACE WOMBO"

Welcome to SPACE WOMBO, a VERY simple and entertaining game where you control a spaceship that must navigate through a field of asteroids and shoot down an alien invader multiple times. 

This README file will provide you with information about the game, how to play, and how to contribute to its development.

![Game Screenshot](screenshot.png)

## Getting Started

### Prerequisites

Before you can play the game, you'll need the following:

- Any computer running Windows, macOS, or Linux.
- python3 & pygame

### Installation

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your-username/spaceship-shooting-game.git
  
2. Navigate to the project directory:
   ```bash
   cd spaceship-shooting-game
3. Open the main.py file.
```bash
   python3 main.py
   
How to Play
Use the arrow keys (or 'Up', 'Down', 'Left', 'Right' keys) to control your spaceship.
Avoid colliding with asteroids by navigating your spaceship.
Shoot down the alien invaders by pressing the spacebar key.
Each time you hit an alien, you'll earn one point.
As your score increases, the game will progressively get more challenging, increasing the speed and density of asteroids and aliens.

Contributing
If you'd like to contribute to the development of this game, please follow these steps:
Fork the repository to your GitHub account.
Clone the forked repository to your local machine.
Make your changes, add new features, or fix bugs.
Commit your changes and push them to your GitHub repository.
Create a pull request from your repository to the original repository.
Your changes will be reviewed and merged if they improve the game.
